# /plan-program
Stand up a new UX program from goal to executable plan.
## Steps
1. **Scope** — Define the problem and boundaries using `scoping-framework` skill.
2. **Roadmap** — Build phases and milestones using `program-roadmap` skill.
3. **Dependencies** — Surface cross-team hand-offs using `dependency-map` skill.
4. **Capacity** — Check the plan against team availability using `capacity-planning` skill.
5. **Prioritise** — Sequence the work using `prioritisation-framework` skill.
## Output
A program plan with scope statement, phased roadmap, dependency map, capacity check, and a prioritised backlog.
Consider following up with `/report-status` once the program is underway.
